# **App Name**: EcoCity

## Core Features:

- Comprehensive Recycling Guides: Detailed information on various waste types, disposal methods, and recycling processes. Includes interactive quizzes and step-by-step disposal guides.
- AI Waste Identifier: AI-powered tool that classifies waste materials through image analysis, providing appropriate disposal instructions. This tool suggests how each classified object should be recycled, reused, or disposed.
- Recycling Center Locator: Uses the Google Maps API to locate nearby recycling centers based on the user's location, displaying accepted waste types and operating hours.
- User Authentication: Secure user registration and login using Firebase Authentication, with support for email/password and Google Sign-In. Enables personalized content and activity tracking.
- Firestore Database Integration: Utilizes Firestore to manage user profiles, marketplace listings, recycling center data, rewards information, and forum posts, ensuring data consistency and scalability.
- Marketplace: Platform for users to buy and sell second-hand items, promoting reuse and reducing waste. Integrated with Firebase Storage for product images and Firestore for product data.
- Rewards Program: Gamified system that rewards users with eco-points for sustainable actions. Eco-points can be earned through community participation, waste classification, recycling efforts. Includes a leaderboard to foster competition and motivation.

## Style Guidelines:

- Primary color: Forest Green (#2E7D32) to represent nature and sustainability.
- Background color: Light Gray (#F5F5F5) for a clean and modern look.
- Accent color: Warm Orange (#FFA726) for CTAs and highlights to draw attention.
- Headings: 'Poppins' sans-serif font for a bold and modern look.
- Body: 'Inter' sans-serif font for clear and readable content.
- Use eco-friendly themed icons to visually represent different waste categories and actions.
- Subtle animations and transitions to enhance user engagement and provide feedback, such as loading animations and hover effects.
